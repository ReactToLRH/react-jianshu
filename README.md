# 简书

本项目是使用`React+Redux`开发。主要包括首页、文章详情页、登录页。

## 如何运行

+ 克隆代码：`https://github.com/ReactToLRH/react-jianshu.git`
+ 安装依赖：`npm install`
+ 运行：`npm run start`
+ 打包：`npm run build`

## 使用插件/模块

+ [styled-components](https://styled-components.com/)：
+ [immutable]
+ [redux-immutable]
+ [react-transition-group]
+ [react-loadable]
